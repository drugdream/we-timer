## 运行

```shell
yarn install
```

### 运行开发模式

```shell
yarn run serve

```

然后使用[微信小程序开发工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/devtools.html)打开 `dist/dev/mp-weixin` 预览

### 构建发布包

```shell
yarn run build
```

发布包内容在 `dist/build/mp-weixin` 文件目录
